


# Todo List Project

------------ דרישות מערכת
- Java 11 או גרסה חדשה יותר
- IDE כגון VS Code או IntelliJ
- קובץ tasks.json ייווצר אוטומטית בהפעלה ראשונה אם לא קיים

-----------איך להריץ את האפליקציה
1. פתחו את הפרויקט ב-IDE שלכם
2. הריצו את הקובץ Main.java
   או דרך המסוף:
   javac Main.java
   java Main

----------פונקציות עיקריות
- הוספת משימה חדשה
- הצגת כל המשימות
- עדכון משימה קיימת
- מחיקת משימה
- סימון משימה כ-DONE
- חיפוש משימות
- הצגת משימות ממוינות לפי סטטוס
- קבלת משימה לפי ID
